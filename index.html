Downloading Games

On your PS Vita, find the PKG application. Click on this application and then click "Start". This will load you into a screen with thousands of games for the PS Vita.

If you press the Triangle button, you can sort these games by title, region, name, size, and date. From this menu, you can also choose to show PSP games or PSX games (PlayStation One games made for PSP).

Click the Circle button on any game you would like to download, and then click "Install Game or Patch". Wait a few seconds and then click the prompt to begin the download. PS Vita games' progress will be viewable from the top right corner on the home screen. PSP and PSX games will show that they are downloading at the bottom of the PKGj app.

If you are downloading PS Vita games, you can close out of this application after you initially begin the download. You will need to keep the application open while downloading PSP and PSX games.

Emulators

mGBA

  • mGBA is an emulator for Gameboy Advance (GBA) and Gameboy Color (GBC) games.
  • Launch the application to access the file section where you can find GBA ROMs and GBC ROMs located at ux0:/data/mGBA/.
  • Controls: Square button (change screen size), Triangle button (access menu).
  • To access the menu: press the Triangle button in the game to bring up the menu. From the menu, you can create and load save states, adjust settings, and exit the game back to the games menu.

CATSFC-libretro-vita

  • CATSFC-libretro-vita is an emulator for Super Nintendo Entertainment System (SNES) games.
  • Press the R button when launching the application to access the ROMs section.
  • Controls: R button (change controls, screen size, or make tweaks).
  • To access the menu: press the R and L buttons together. From the menu, you can create and load save states, adjust settings, and perform other actions.

DaedalusX64

  • DaedalusX64 is an emulator for Nintendo 64 (N64) games.
  • Wait for approximately 30 seconds for the application to load.
  • Controls: Left analog stick (cursor), L button (left click), R button (right click), D-pad (scroll through games).
  • To access the menu: press the Select button to bring up the top option bar. Use the left analog stick to click on the "File" option with the L button. From the menu, you can create and load save states, adjust options, and close the ROM.

GenesisPlusGXVITA

  • GenesisPlusGXVITA is a Sega Genesis emulator.
  • When launching the application, press the R button to access the ROMs section.
  • Controls: R button (access ROMs or change settings).
  • To access the menu: press the R and L buttons together. From the menu, you can create and load save states, adjust settings, and perform other actions.

PNES

  • PNES is an NES emulator.
  • Upon opening the application, all available games will be listed on the left side.
  • If any games are written in red, press the Start button and change the "Show" option to "Available".
  • Controls: Select and Start buttons (open menu).
  • To access the menu: press the Select and Start buttons together. From the menu, you can create and load save states, and perform other related actions using the "States" button at the bottom.

Adrenaline

  • Adrenaline is a PSP emulator.
  • Go to the Game tab, scroll to the bottom, and click on "Memory Stick" to access PSP games.
  • Controls: Hold the PS button (bring up Adrenaline's menu).
  • To access the menu: click on the Settings button at the top of Adrenaline's menu. From there, you can exit the PSPEmulation or Adrenaline, and perform other actions. Pressing the R button allows you to create or load save states.

VitaDB Downloader

  • VitaDB Downloader is an application used for downloading homebrew games and applications.
  • Note that most of these applications are fan-made games or ports.
  • While browsing, you may find interesting applications or even games like Flappy Bird.
  • Be aware that the Android games section will not work.
  • Explore the available options and discover cool applications!
  • Accessing the menu: N/A

VitaShell

  • VitaShell is a file manager for the PS Vita.
  • Pressing the Triangle button brings up the side menu, where you can copy or move files if needed.
  • Pressing the Select button allows you to connect to a computer and add ROMs to the ROM folders if desired.
  • Accessing the menu: Triangle button (side menu), Select button (connect to a computer).
